⚙️ Groq API Key Setup

Groq is completely free — no credit card needed.

  1. Go to console.groq.com
  2. Sign up with your email
  3. Click API Keys → Create API Key
  4. Copy the key (starts with gsk_)
  5. Paste it below and click Save

🔎 Serper API Key Setup (Source tab)

Serper does real Google searches — used by the Source tab to find actual URLs instead of letting the AI guess. Free tier: 2,500 searches.

  1. Go to serper.dev
  2. Sign up with Google
  3. Copy your API key from the dashboard
  4. Paste it below and click Save

Your keys stay on your device. Stored locally, sent only to Groq and Serper directly.
